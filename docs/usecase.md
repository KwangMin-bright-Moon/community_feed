# **유저 메인 서비스 FLOW**

1. **유저 생성**
    - 유저는 id 를 통해 구분이 되어야 함
    - 이름, 프로필 이미지를 입력 받으면 저장되어야 함
        - 단, 이름은 빈 값이 입력되면 안됨
2. **팔로우 기능:**
    - 사용자는 다른 이용자를 팔로우 할 수 있음
    - 팔로우 취소 할 수 있음
    - 자기 자신을 팔로우 할 수 없음
3. **프로필 노출:**
    - 사용자 프로필을 누르면 다음과 같은 정보들이 노출 됨
        - 이름, 유저 프로필, 팔로잉 숫자, 팔로워 숫자
        - 팔로잉 팔로우 유저 리스트
            - 이름, 프로필
            - 단,  팔로잉, 팔로우 숫자를 클릭 했을 때에 각 유저 리스트가 노출됨
